# Research

Use this skill when the task requires discovery, comparison, synthesis, or evidence.

## Workflow

1. Define the question.
2. Identify sources or local evidence.
3. Separate facts from assumptions.
4. Compare alternatives.
5. Produce a concise recommendation.
6. List confidence and unknowns.

## Output Format

- Summary
- Key findings
- Recommendation
- Risks
- Next steps
